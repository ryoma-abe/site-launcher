# Site Launcher

Site Launcher は、よく使うサイトをショートカットキーでぱっと開ける Chrome 拡張機能です。

## できること

- Cmd+M / Ctrl+M などのキーでランチャーを表示
- A〜Z・0〜9 の 1 文字を割り当ててサイトを素早く開く
- 右クリックメニューから開いているページを即座に登録
- 設定ページで一覧編集・ショートカット変更・バックアップが可能
- JSON 形式でエクスポート / インポートし、別端末へ移行

## インストール方法

1. Chrome ウェブストア版（公開済みの場合）をインストール。または GitHub Releases の ZIP をダウンロード。
2. 手動インストールする場合は `chrome://extensions/` を開き、デベロッパーモードをオン → 「パッケージ化されていない拡張機能を読み込む」から `dist` フォルダを選択します。

## 使い方

1. ツールバーアイコン、またはショートカットキーでランチャーを開きます。
2. 登録済みサイトをクリックするか、割り当てたキーを押すと開きます。
3. 「新しいサイトを追加」フォーム、あるいは右クリックメニューからサイトを登録できます。
4. 設定ページでは一覧の編集・削除、ショートカット重複チェック、エクスポート / インポートが行えます。
5. ナビゲーションの「説明ページ」ボタンからブラウザ版ガイドを閲覧できます。

## バックアップと移行

1. 設定ページ右上の「エクスポート」で JSON ファイルを保存。
2. 別端末で読み込んだ拡張に対し、「インポート」からファイルを選択すると復元できます。
3. フォーマットが不正な場合はエラー表示となり、既存データはそのままです。

## よくある質問

- **ショートカットは何個まで？** A〜Z と 0〜9 の 36 個までです。
- **同期は？** Chrome の同期ストレージを利用しているため、同じ Google アカウントでログインしていれば自動同期されます。
- **バックアップ JSON に機密情報は？** URL と表示名、割り当てキーのみを保存します。
