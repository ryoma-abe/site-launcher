# Site Launcher

Site Launcher は、よく使うサイトへキーボードですばやくアクセスできる Chrome 拡張機能です。ポップアップからのワンクリック起動はもちろん、右クリックメニューやショートカットのカスタマイズ、バックアップ機能にも対応しています。

## 主な機能

- **クイックランチャー**: Cmd+M / Ctrl+M などのショートカットでランチャーを表示
- **サイト管理**: ポップアップや設定ページからサイトを追加・編集・削除
- **ショートカットキー割り当て**: A〜Z / 0〜9 の1文字でサイトを登録
- **右クリック追加**: ページやリンクをコンテキストメニューからワンタッチで登録
- **設定ページ**: 一覧表示、インポート / エクスポート、編集をまとめて操作
- **バックアップ**: JSON 形式でサイト一覧を保存し、別の環境へ移行可能

## インストール

### Chrome ウェブストア
公開準備中です。公開後はここにリンクを掲載します。

### 手動インストール（ベータ版）
1. GitHub の [Releases](https://github.com/ryoma-abe/site-launcher/releases) から最新版の「Build」アーカイブをダウンロードします。
2. ZIP を解凍し、`dist` フォルダを手元に用意します。
3. Chrome で `chrome://extensions/` を開き、右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」から `dist` フォルダを選択します。

> **ヒント:** 更新時は同じ手順で `dist` を読み込み直すだけで OK。Chrome 側で既存データは保持されます。

## 使い方ガイド

### ポップアップからサイトを開く
1. 既定のショートカット（Cmd+M / Ctrl+M）またはツールバーアイコンでランチャーを開きます。
2. 登録済みのサイトをクリック、もしくは割り当てた1文字キーを押して開きます。
3. 右上のナビゲーションボタンから設定ページや説明ページへ移動できます。

### サイトを追加する
- ポップアップ: 「新しいサイトを追加」フォームに *サイト名 / URL / ショートカットキー* を入力して登録
- 右クリック: ページ上で右クリック → 「Site Launcher に追加」 → 設定ページが開くので内容を確認して保存

### 設定ページでできること
- サイト一覧の編集・削除、ショートカットの重複チェック
- JSON ファイルへの **エクスポート** / JSON からの **インポート**
- Chrome のショートカット設定画面へのリンク

詳しい手順は拡張内の「説明ページ」ボタン、または [docs/import-export.md](docs/import-export.md) を参照してください。

## バックアップと移行

1. 設定ページ右上の「エクスポート」を押して JSON ファイルを保存します。
2. 別環境で拡張を読み込んだ後、「インポート」からファイルを開くと登録内容が復元されます。
3. 右クリックメニュー経由で追加したサイトも同じファイルに含まれます。

> **注意:** JSON の形式が変わった場合は拡張側で自動的に整形されますが、手動で編集する際は `name` / `url` / `key` を必ず含めてください。

## トラブルシューティング

| 症状 | 対処方法 |
| --- | --- |
| サイトを開けない / 404 になる | URL の冒頭に `https://` が付いているか確認してください。登録時に自動補完されますが、独自ポートなどは手動入力が必要です。 |
| ショートカットが反応しない | Chrome のショートカット設定で他の拡張と重なっていないか確認し、必要に応じて変更してください。 |
| インポートでエラーになる | JSON ファイルが破損していないか、`sites` 配列が空でないかを確認してください。詳細は説明ページの「トラブルシューティング」を参照。 |
| 右クリックメニューに表示されない | 拡張を一度無効化→有効化するとメニューが再登録されます。Chrome を再起動すると確実です。 |

## よくある質問

- **ショートカットは何個まで登録できますか？** 36個（A〜Z / 0〜9）までです。重複すると設定ページで警告が表示されます。
- **同期はできますか？** Chrome の同期ストレージを利用しているため、同じ Google アカウントでサインインしていれば自動同期されます。
- **バックアップファイルは暗号化されていますか？** JSON 形式で保存されます。機微情報は含まれませんが、取り扱いにはご注意ください。

## プライバシー

- 登録したサイト情報は Chrome の同期ストレージにのみ保存され、外部サーバーには送信されません。
- ファビコン取得には Google Favicon API を利用します（送信するのはドメイン名のみ）。
